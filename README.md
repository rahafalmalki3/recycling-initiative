# Recycling initiative Webpage
Frontend web project designed to encourage uniform recycling.
**[View Live Demo Here](https://rahafalmalki3.github.io/recycling-initiative/)**

## Project Structure 
* `index.html` - Main landing page.
* `locations.html` - Detailed list of recycling drop-off points.
* `css/style.css` - Stylesheet utilizing CSS Variables for theming.
* `assets/` - Directory for project images.
